Set up Opera for secure browsing

PUT IN TIME TO COMPLETE SET UP AND COST

Opera is a well designed and highly functioning web browser with hundreds of millions of users worldwide, built on the Blink engine from Chromium. Many features now common to all web browsers, such as tabs, private browsing and pop-up blocking, originated by Opera. The simple layout, extensive functionality and built-in VPN service make it a great candidate for browsing the web safely and securely. This tutorial will help you set up the essentials to help keep your personal data under your control.

TL;DR

  1. Download and install Opera
  2. Click the VPN icon in the address bar and make sure it's on
  3. Settings > Privacy and Security
  • check show advanced settings
  • check Block ads and Send a ‘Do Not Track’ request with your browsing traffic
  • uncheck autofill and save passwords
  • WebRTC > Disable non-proxied UDP
  1. Install HTTPS Everywhere
  2. Install Privacy Badger

Get Opera

If you haven't already, download and install Opera. The set up process is very quick, but does require agreement to terms and a privacy policy. I recommend taking a look at the privacy policy to make sure you really agree.

Opera Settings

The settings can be easily accessed in any of the following ways:

  • The left-hand navigation bar on Speed Dial
  • The gear icon at the top right in Speed Dial (click Go to browser settings at the bottom)
  • Menu > settings
  • alt + p
Before we walk through the settings, go to the Privacy and security tab, scroll to the bottom and check the option Show advanced settings
Browser tab
  1. Search settings: switch search engine to DuckDuckGo
  2. User interface: check the option Show the full URL in combined search and address bar. This allows you to always easily see exactly what site you're on.
  3. Battery saver: this one isn't a security issue, but if you're on a laptop, the battery saver is a good idea.
Websites tab
  1. JavaScript: for stricter security, choose Do not allow any site to run JavaScript. However, this may break many websites. If you would like to have functioning websites more than strict security, leave this option alone.
  2. Location: choose the ask me ... option.
  3. Notifications: choose the ask me ... option.
  4. Microphone: choose the ask me ... option.
  5. Camera: choose the ask me ... option.
  6. MIDI full control: choose the ask me ... option.
Privacy and Security tab
  1. Block ads: check the box. (This blocks many malicious ads which track your behavior across websites)
  2. Privacy: only Protect me from malicious sites and Send a ‘Do Not Track’ request with your browsing traffic should be checked
  3. VPN: check the box to enable the VPN. What's a VPN?
  4. Autofill: uncheck this box.
  5. Passwords: uncheck this box. Instead use an independent password manager like LastPass or KeePassX.
  6. WebRTC: choose Disable non-proxied UDP. This can stop the built-in VPN from leaking your real IP address.
  7. Cookies: the recommendation here is to choose Allow local data to be set, and check block third-party cookies and site data. Don't worry too much about this one, we'll cover cookie-blocking in just a moment.

Adding more security with extensions

  1. Get HTTPS Everywhere. This extension forces Opera to get the encrypted version of any website that has one.
  2. Get Privacy Badger. Go through the intro page to learn how Privacy Badger works.

Now, in Opera, go to opera://extensions/. Find HTTPS Everywhere and check the box Allow in private mode. Do the same for Privacy Badger.
